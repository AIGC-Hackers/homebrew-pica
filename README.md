# homebrew-pica

Homebrew tap for the Picadabra CLI.

## Install

```bash
brew tap AIGC-Hackers/pica
brew install pica
```

The formula installs the lightweight JS bundle and declares Bun as a runtime
dependency automatically.

## Upgrade

```bash
brew update
brew upgrade pica
```

## Notes

- The `pica` formula in this tap is managed automatically by the upstream
  release workflow.
- Release artifacts are published at
  <https://github.com/AIGC-Hackers/pica-cli/releases>.
